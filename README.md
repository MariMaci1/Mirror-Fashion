<h1>🛍️ Mirror Fashion</h1>

<h2>O Mirror Fashion é um site de e-commerce de moda, desenvolvido para consolidar conhecimentos sólidos em estruturação web semântica, estilização avançada e design responsivo.</h2>

<h3>🚀 Tecnologias Utilizadas</h3> 
<p>
* HTML5: Utilização de tags semânticas para melhoria de acessibilidade e SEO.<br>
* CSS3: Layouts modernos utilizando técnicas de posicionamento, estilização de componentes, tipografia e<br> transições suaves.<br>
* Boas Práticas:Organização de arquivos, nomenclatura de classes legível e foco na fidelidade do layout proposto.
</p>


<h3>🧠 O que aprendi neste projeto</h3>
<p>
* Semântica Estrutural: A importância de construir uma base HTML sólida para acessibilidade e motores de busca.<br>
* Modularização em CSS: Como reutilizar estilos e manter uma folha de estilo organizada e de fácil manutenção.<br>
* Resolução de Problemas: A experiência na coordenação de equipes pedagógicas me ajudou diretamente na<br> organização das etapas de desenvolvimento deste projeto, dividindo o design em partes menores e solucionando<br> desafios de layout de forma lógica e estruturada.<br>
</p>
